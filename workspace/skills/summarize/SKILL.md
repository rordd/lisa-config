# Summarize Skill

## Usage

1. Fetch the page: `web_fetch(url, extractMode="markdown")`
2. Summarize the content in Korean

## Rules

- Summarize in Korean
- Extract 3-5 key points
- Keep it concise even for long articles
- Include code examples for technical docs if relevant
- Mention the source URL
